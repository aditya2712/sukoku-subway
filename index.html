Sudoku:

Sudoku is a logic-based, combinatorial number-placement puzzle. In classic sudoku, the objective is to fill a 9×9 grid with digits so that each column, each row, and each of the nine 3×3 subgrids that compose the grid contain all of the digits from 1 to 9.

Sudoku Solver:

Just plug-In the values that are given in your grid, in the grid shown, and don't worry about the rest cells (Leave them empty). Now if you wish to get a hint at any random position then hit the Get Random Hint button or if you want the sudoku to be solved then hit the Solve button.
